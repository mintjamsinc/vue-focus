# vue-focus
A reusable focus directive for Vue.js 2.x.

## Installation

```sh
npm install --save-dev @mintjamsinc/vue-focus
```

## Usage

```js
import VueFocus from '@mintjamsinc/vue-focus';
Vue.use(VueFocus);
```

```vue
<input type="text" name="id" v-model="id" v-focus.delay="200">
```
